**Home SOC Environment**

## Objective

In this lab, I established a comprehensive Security Operations Center (SOC) environment within a cloud platform to simulate and detect cyber threats, as well as to execute appropriate responses to alerts generated by security incidents. The primary objective was to analyze and respond to various attack scenarios across Windows and Linux systems using a systematic ticketing process. This practical experience was intended to enhance understanding of network security principles, attack methodologies, defensive techniques, and real-world incident response workflows.

### Skills Learned

- In-depth knowledge of SIEM concepts and their practical application.  
- Strong proficiency in analyzing and interpreting network logs.  
- Ability to identify and recognize attack signatures and patterns.  
- Comprehensive understanding of network protocols and security vulnerabilities.  
- Development of critical thinking and problem-solving skills within cybersecurity.

### Tools Used

- **Elastic SIEM** and **Kibana** for log ingestion and analysis.
- **Sysmon** integration to get deep, security-focused telemetry
- **Mythic Agent** for C2 channel and data exfiltration.
- **osTicket** integration to generate tickets for alerts.
- Automate remmediation action for endpoint device using **Elastic Defend**.
